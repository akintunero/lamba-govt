# Security Policy

## Supported Versions
| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

**Please Note:** The GovLamba Platform is an intentionally vulnerable Capture The Flag (CTF) environment. Do **not** report the built-in OWASP vulnerabilities (e.g., SQL Injection, IDOR, SSRF) as security issues. 

However, if you discover a vulnerability in the underlying lab infrastructure that allows for unintended container escapes, host-machine compromise, or flaws in the Docker Compose configuration, please report it immediately.

**Contact:**
Report infrastructure vulnerabilities directly to Olúmáyòwá Akinkuehinmi at **akintunero101@gmail.com**. Please allow 48 hours for a response before disclosing the infrastructure flaw publicly.
